Dataset and Annotations for Counting in Extremely Dense Crowds
--------------------------------------------------------------

Contains 50 images used in the following paper:

Haroon Idrees, Imran Saleemi, Cody Seibert, Mubarak Shah,
Multi-source Multi-scale Counting in Extremely Dense Crowd Images,
In IEEE Conference on Computer Vision and Pattern Recognition (CVPR), Portland 2013.

For experiments, 5-fold cross validation was performed by randomly selecting images in each fold.
